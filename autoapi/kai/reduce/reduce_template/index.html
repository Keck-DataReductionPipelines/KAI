kai.reduce.reduce_template.jackknife()[source]

Perform the Jackknife data reduction. The methodology is as follows: For N total images, there are N stacked images created by data.combine(), each of these combo frames consists of N-1 images. The suffix ‘_{i}’ is given to each jackknife combo frame (i.e. mag27maylgs_12_ks.fits would be the 12th jackknife combo frame created). Submaps flag has been turned off for this function, as it is not needed.
