kai.reduce.reduce_template_osiris.jackknife()[source]

Perform the Jackknife data reduction. The methodology is as follows: For N total images, there are N stacked images created by data.combine(), each of these combo frames consists of N-1 images. The suffix ‘_{i}’ is given to each jackknife combo frame (i.e. mag_OB06284_17_kp_tdOpen.fits would be 17th jackknife combo frame created). Submaps flag has been turned off for this function, as it is not needed.
